## Project Outlines

- Binary Counter ([docs](/docs/binary-counter.md)/[repo](https://github.com/mdemark18/SAMD-Binary-Counter))</br>
This binary counter was built for Montana Tech's Electrical Engineering Department to use as a student outreach tool. The PCB was designed by Bryce Hill; I wrote the firmware in C for the SAMD ARM microcontroller to his specifications. The counter displays a binary count from 0–1023 on an LED bar, and leverages the chip's built-in UART to allow users to connect via serial and control it from a terminal. I also wrote the supporting documentation and developed a companion testing board to simplify the flashing and validation process.
- Pick-and-Place Deployment ([docs](/docs/pick-and-place.md)/[repo](https://github.com/mdemark18/pick-and-place))<br>
Our department purchased a pick-and-place machine shortly after I became a student employee here and this specific machine came with little to no documentation on how to run it. I was in charge of getting the machine up and running, providing documentation, and a workflow on how to use it as efficiently as possible. 
- Miller’s Games ([docs](docs/millers-games-doc.md)/[website](https://millersgames.free.nf/index.php)/[repo](https://github.com/mdemark18/WebProgramming25))</br>
This is a website I created for my Web Programming course in Spring 2025. This site is a functional game marketplace much like BestBuy’s site or Gamestop’s. All features are functional, short of actually having stock and shipping it to you! The website is currently hosted through InfinityFree and has a default site name. Everything else is as seen in my code within my repository.
- Homelab Hospital Network ([docs](docs/homelab-hospital-doc.md)/[repo](https://github.com/mdemark18/Homelab-Hospital-Network))</br>
This is a custom network setup I created using a 4 port NIC, 4 virtual machines, a Cisco Router, and a Cisco Switch to emulate a hospital network and how different VLANs would function together. This is further tied together with the use of open source software to fully emulate the computers and software that may be used between departments.
- ESP32 Fish Tank Monitor (*In Progress*) ([docs](docs/tank-monitor-doc.md)/[repo](https://github.com/mdemark18/Tank_Monitor))</br>
This is a current project designed to create a fish tank monitoring solution. As an avid aquarium keeper, this will save me headaches by tracking water temperature and level on an ESP32 and sending that data from my multiple tanks to a central server, allowing me to quickly check and receive alerts for critical tank levels. This includes custom PCBs and future 3D printed housings for the monitors. More features to come as I tinker.
